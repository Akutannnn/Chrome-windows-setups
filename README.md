# Chrome Windows Setups  

## What is Chrome Windows Setups?  

Chrome Windows Setups is a browser extension that saves and restores your entire Chrome workspace with one click.  

It is basically a tool to save and load Chrome windows and tabs disposition.

**Perfect for:**  
- Developers who switch between different projects with specific tabs and windows layouts  
- Students managing separate windows for different courses or research topics  
- Traders working with multiple charts using multiple Chrome Windows  
- Anyone who wants to quickly restore their browsing sessions without losing window positions  

## How It Works

1. Open the extension's popup by clicking on its icon.  

You should see that :  
<img src="/images/readme/firstuse.png" alt="image1" width="450" />  

2. Enter a name (or not) for your current tabs and windows disposition and press "Save Current Setup"  

3. Your setup name will appear right below. 

4. Press load on a setup to open up the associated saved windows and tabs disposition.

## Features  

<img src="/images/readme/setupsexample.png" alt="image1" width="450" />  

- "Load" button opens the setup.  
- "Save" button overwrites the setup save with the current tabs and windows disposition. (When pressed it will show a confirmation popup. You can check "Don't show this again." to remove the confirmation popup for the future.)
- "Delete" button deletes the setup.
- Works with multiple screens (double screen, triple screen etc.)
- Save Current Setup defaults to "Setup 1", "Setup 2"... if no name is entered.  

**What gets saved**
- All open windows and their positions.
- All tabs in each windows with their URLs and order
- Pinned tabs

## Known issues

Minimized windows are not saved.  
The extension can't get position or size information for minimized windows which are necessary when loading a window.  
Only non-minimized windows (normal, maximized, or fullscreen) will be included in saved setups.
